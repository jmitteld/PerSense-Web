Printing the schedule
Now press Ctrl+T to generate a table. A Table Output Options window opens offering you several choices:
  • How do you want the table summarized? You can print a summary line at the end of each quarter or year (DETAIL & SUMMARY), you can print just the quarterly summaries without lines for the individual payments (SUMMARY ONLY), or you can leave out the summary altogether (DETAIL ONLY).
  • In what month(s) do you want the summary triggered? If you elected a summary option in line 1, you must specify whether, for example, the semi-annual summaries are printed in June and December or in January and July.
  • Where should the table print? It can be routed to the screen, or a file. You can format the file as a word processor file (plain text) or as a comma seperated file, useful for importing it into most spreadsheet programs.
Click OK when the options in the window are correct. The table will print.
