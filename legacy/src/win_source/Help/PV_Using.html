Using the Present Value Screen for Loan Calculations
Because the concept of present value is closely tied to loan and savings computations, you can calculate them on this screen as long as a single interest ate prevails throughout the time period.

Usually, loan computations on the Present Value screen won't provide ccuracy to the penny. This is a general-purpose screen: there is no setting for Prepaid interest, Interest in advance or arrears, USA Rule, or Treatment of partial interest periods. For the most straightforward loan schedules, in which payments begin just one month after closing and continue monthly thereafter, the Present Value screen will reproduce the results of the Amortization screen exactly. For more complex situations, small discrepancies (generally a dollar or two in a thousand) will result. Occasionally, the differences can be larger.
