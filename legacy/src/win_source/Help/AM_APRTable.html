Creating an APR Table
To generate an amortization table from a screen defining a loan, press Ctrl-T , or select Table output from the Calc menu. If sufficient data has been entered, a window will open offering output options for the table you have requested.

Output options for table Detail lines & Summary: SUMMARY ONLY Summary period: ANNUAL Summary in which months? 12

The first line in this window offers three options: DETAIL ONLY, DETAIL & SUMMARY, and SUMMARY ONLY. PerSense creates one detail line for each date on which a payment occurs. A summary line totals several detail lines, for example, all the payments in a given calendar year. Choose a table with detail lines only, summary lines only, or both.

If you choose DETAIL & SUMMARY or SUMMARY ONLY, PerSense wants to know how frequently to summarize.

Next, tell PerSense for which months to generate summary lines.
  • Yearly summaries can be in any of the 12 months. Enter a number from 1 to 12. If your fiscal year ends in March, for example, you'd enter a 3 here.
  • Semi-annual summaries can be done for any combination of months six months apart - every June and December, for example. If you enter 6 (for a June summary line), PerSense enters 12 beside it.
  • Specify quarterly summary months in the same way: enter the first month in which you want a summary to print, and the other three are supplied.
Hint: Most amortization tables are created just to obtain a single number. With PerSense's Balance as of feature, you may never need to print a table. Simply enter the As of date in the lower right of the screen, and press Enter : the remaining principal balance on that date will be displayed next to it.
