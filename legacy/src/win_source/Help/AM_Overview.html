The Amortization Screen
The Amortization screen is a flexible loan analysis tool. Use it to create amortization schedules, or just to compute payment amounts for a complex, structured loan.

An amortization table (or schedule) is a list of loan payments, in which each payment is divided as interest and principal parts. Per%Sense amortization tables also include summary data: the total interest paid to date, the remaining balance of the loan, etc.

To get to the Amortization Screen you have a number of options:
  • Select the "Screens" menu and choose "Show Amortization"
  • Right click on the background to bring up the context menu and select "Show Amortization"
  • Press Alt+A
