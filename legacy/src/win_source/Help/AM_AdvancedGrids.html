Grids in the Advanced Amortization Screen
With Extended Options turned on, the Amortization screen is a powerful loan analysis system, capable of computing the most complex of today's creative financing schemes. Press X to open the Extended Options. The advanced amortization screen contains the following grids
  • Periodic Payments Grid
  • Balloon Payments Grid
  • Adjustments Grid
  • Moratotium Grid
  • Target Grid
  • Skipped Months Grid
