Appearance Settings

Cell background color

This is the color of the background of all the usable cells in a Per%Sense grid.

Highlighted cell background color

This is the color of a cell that is currently selected.

Calculated cell background color

This is the color of a cell that contains soft data or data that Per%Sense calculated for you.

Cell font

The font for cells that contains hard data or data that you entered yourself.

Calculated cell font

The font for cells that contain soft data or data that Per%Sense calculated for you. Note that Per%Sense will automatically show this font in bold.
