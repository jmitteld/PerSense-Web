Clipboard Settings
Clipboard settings allow you to change how Per%Sense stores data when it is copied or pasted to the clipboard, and how it reads data that is pasted from the clipboard. This makes it possible to cut data from a Per%Sense data sheet and then paste into some other spreadsheet program, or to cut from a spreadsheet and paste into Per%Sense.

Cell delimiter

This is the character that Per%Sense will put between each cell. Some standard options are available in the drop down, or you can type in whatever you want. The two special case options are "{tab}" and "{CR+LF}" which are meant to represent the tab key, and a carriage return and line feed respectively.

Row delimiter

This value is the same as the cell delimiter, only it comes at the end of each row of data.
