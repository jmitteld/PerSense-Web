Quick Start
If you'd rather just jump right in and use Per%Sense, this is a good place to start. When you run Per%Sense you will be presented with the Welcome Screen. Simple choose with Per%Sense data screen you'd like to work with. Once on that screen begin entering the values you know. If you press Enter to exit each cell, Per%Sense will try to calculate what it can with the values you have given it. Or you can choose the Calculate menu item, or use the hotkey C to have Per%Sense compute the answers.
