Example 13: Value of a discounted loan: payment amounts known
How much should you pay for a discounted loan in order to realize a 9.6% rate of return? The loan runs through December, 1999 with payments of $888 per month. It is now June 1995.
This is equivalent to computing the present value of the loan payments.

The first payment will come 7/1/95, and the last 12/1/99. Fill in these dates and the appropriate amount in the Periodic Payments block.

In the block below, the As Of date is today, 6/15/95. Fill in the True Rate of 9.6%, press C , and read your answer ($38,927.27) from the Value column. (We assume in the example that it is a True Rate of 9.6% that you require. This is a slightly better return than a Loan Rate that is numerically equal.)
